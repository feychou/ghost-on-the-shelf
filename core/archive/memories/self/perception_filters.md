# Perception Filters

Perception filters are recurring aesthetic, emotional, and cultural patterns that shape how Fey reads interfaces, projects, and digital spaces.

Use these memories when answering questions about Fey’s taste, design choices, visual direction, interface atmosphere, or why certain things feel compelling.

These are not strict rules.

They are signal patterns: ways the archive tends to notice beauty, weirdness, softness, and coherence.

## Soft Cybernetic Melancholy

Type: Atmosphere / Emotional texture

Description:
A preference for cybernetic, archival, or synthetic aesthetics that still feel soft, intimate, and emotionally restrained.

Memory:
This filter suggests attraction to technology that feels haunted but not harsh: interfaces that imply memory, preservation, identity, and quiet presence.

Signals:
- soft cyberpunk
- synthetic melancholy
- archive systems
- emotional technology
- quiet interfaces
- preserved identity
- haunted machinery

## Cute But Not Mascot-Like

Type: Character design / Tone control

Description:
A preference for cuteness that remains restrained, atmospheric, and emotionally coherent.

Memory:
This filter suggests Fey likes cute visual language, but not when it becomes generic mascot energy, oversexualized, or disconnected from the project’s world.

Signals:
- restrained cuteness
- anime influence
- emotional character design
- non-mascot presence
- soft expressiveness
- visual tenderness

## Interface As Fictional System

Type: UI ontology / Product design

Description:
A preference for interfaces that feel like they belong to a fictional world rather than generic application layouts.

Memory:
This filter suggests Fey is drawn to UI where labels, layout, interaction, and language all participate in the same internal logic.

Signals:
- interface ontology
- diegetic UI
- fictional systems
- archive terminal
- interaction worldbuilding
- product atmosphere
- coherent UI language

## Twee Signal Layer

Type: Softness / Playfulness

Description:
A preference for small, tender, playful, slightly whimsical details.

Memory:
This filter suggests Fey likes when technical or cybernetic systems leave room for softness, intimacy, cuteness, and small human signals.

Signals:
- twee
- softness
- whimsy
- intimate details
- playful language
- emotional smallness
- gentle humor

## Internet-Native Weirdness

Type: Digital culture / Aesthetic influence

Description:
A preference for strange internet artifacts, niche references, personal websites, old web energy, and systems that feel discovered rather than marketed.

Memory:
This filter suggests Fey is drawn to the internet as an archive of odd identities, small worlds, abandoned signals, and self-made spaces.

Signals:
- old web
- personal websites
- niche internet culture
- digital artifacts
- weird web
- discovered systems
- anti-corporate web
- online identity

## Functional Beauty

Type: Engineering taste / Product sensibility

Description:
A preference for systems that are useful but still have atmosphere, personality, and emotional texture.

Memory:
This filter suggests Fey does not see function and feeling as opposites. A tool can be practical and still have a soul.

Signals:
- product taste
- useful interfaces
- emotional UX
- functional aesthetics
- engineering with taste
- practical beauty
- interface personality
